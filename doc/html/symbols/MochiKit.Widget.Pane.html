Class MochiKit.Widget.Pane


Extends MochiKit.Widget.
The pane widget class. Used to create the simplest form of element container. It is also used inside various types of paged containers, such as a TabContainer, a Wizard and similar. A pane only uses a <div> HTML element, and supports being hidden and shown according to any page transitions required by a parent container. In addition to standard HTML events, the "onenter" and "onexit" events are triggered whenever the pane is used in a parent container page transition.
